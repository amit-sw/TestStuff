AI Club Gym (Client-Side Set)

This local bundle recreates the five AI Gym activities that are browser-only. Each demo is a single HTML file, with no back-end dependency required for the core activity.

1. Object Detection CLIENT ONLY

Upload an image and detect objects using COCO-SSD in-browser inference.

File: object-detect.html

Open Gym

2. Fit Lines CLIENT ONLY

Place points on a graph and fit a least-squares regression line instantly.

File: fit-lines.html

Open Gym

3. Clustering Visualization CLIENT ONLY

Create or sample point clouds and run k-means clustering directly in the browser.

File: clustering-visualization.html

Open Gym

4. Handpose Detection CLIENT ONLY

Stream webcam input and draw hand landmarks using TensorFlow Handpose.

File: handpose-detect.html

Open Gym

5. Emotion Detection CLIENT ONLY

Estimate facial expression probabilities from webcam input with face-api.js.

File: emotion-detect.html

Open Gym
